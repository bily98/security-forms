# Security WPF

## Description

This package provides base classes to add role-based authentication for WPF Windows.

## Getting Started

### Dependencies

* Net 6.0

### Installing

* Package Manager
```
Install-Package Security.Wpf -Version 1.0.2
```

* Net CLI
```
dotnet add package Security.Wpf --version 1.0.2
```
