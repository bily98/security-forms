# Security Forms

## Description

This package provides base classes to add role-based authentication for DevExpress Forms.

## Getting Started

### Dependencies

* DevExpress 21.2

### Installing

* Package Manager
```
Install-Package Security.Forms -Version 1.0.0
```

* Net CLI
```
dotnet add package Security.Forms --version 1.0.0
```
